# Examples Module

## Purpose
This directory contains example applications demonstrating how to integrate and use the Grape terminal emulator library.

## Key Files
- `demo.cpp`: A complete, headless data renderer test that uses OpenGL 1.1 fixed-pipeline rendering to draw the terminal quads generated by Grape.

## Public API
*Internal module — no public interface.*

## Dependencies
- **Grape**: Uses `grape.h` (`GRAPE_IMPLEMENTATION`).
- **GLFW3**: For window creation and input handling.
- **OpenGL**: For uploading the texture atlas and rendering the quads.

## Usage Examples
The `demo.cpp` file initializes a window, creates a `grape::Terminal` instance, and handles the main event loop:
```cpp
// Setup terminal
grape::Config cfg;
grape::Terminal terminal(80, 24, cfg);

// Upload font atlas to GPU
const auto& atlas = terminal.getAtlas();
// ... glTexImage2D ...

// Main loop
while (!glfwWindowShouldClose(window)) {
    // Handle input & resize...
    if (terminal.hasChanged()) {
        const auto& quads = terminal.getQuads(0, 0);
        // Draw quads using OpenGL
    }
}
```
